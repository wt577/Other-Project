							 Quad GCS 实验平台使用说明书

 第一章：软件启动与连接
1.1 硬件准备
台架环境：确保四旋翼无人机已牢固安装在单轴/三轴测试台架上，活动范围无障碍物。
飞控连接：通过 USB 或数传连接飞控到电脑（推荐 USB，波特率更稳定）。若使用 RflySim 仿真，请确保 CopterSim 已启动。
1.2 建立通信
启动软件：双击运行 control.py。
配置参数（左侧“UDP 节点配置”卡片）：
本地端口 (Local Port)：默认为 14550（PX4/RflySim 标准端口）。
目标 IP (Target IP)：
实物飞控：通常为 127.0.0.1（若通过 MAVLink 转发）或飞控的局域网 IP。
RflySim 仿真：默认为 127.0.0.1。
目标端口 (Target Port)：默认为 14555。
点击 [启动连接]：
✅ 成功标志：左上角指示灯变绿，状态显示 LINK OK，电压/姿态数据开始跳动。
❌ 失败排查：检查端口是否被 QGroundControl 或 MissionPlanner 占用，请先关闭其他地面站。
🎮 第二章：飞行控制与安全解锁 (核心)
⚠️ 警告：此环节涉及电机旋转，请务必保持安全距离！

本软件采用“双重安全锁”机制，防止误触导致电机意外旋转。

2.1 解锁流程 (Standard Arming Sequence)
第一步：允许发送指令
勾选左侧 [1] 允许发送姿态指令。
此时软件开始向飞控发送 Attitude Setpoints（默认为 0 度，油门 0），但电机不会旋转。
第二步：切换 OFFBOARD 并解锁
勾选 [2] 允许解锁电机 (ARM)。
现象：飞控模式切换为 OFFBOARD，电机开始怠速旋转（或根据飞控设定停转）。
注意：若无法解锁，请检查：
电池电压是否过低。
遥控器（若有）油门是否已拉至最低。
飞控是否报“Mag/Accel Calibration”错误。
2.2 飞行控制 (Setpoints)
在左侧 “飞行控制 (SETPOINTS)” 卡片中，你可以手动控制无人机：

THR (油门/推力)：拖动滑块控制电机转速。
起飞操作：缓慢向上拖动 THR 滑块（范围 1100~1900 PWM）。
悬停点：通常在 1450~1550 左右（视机型而定），此时无人机应在台架上浮起或拉紧绳索。
Roll / Pitch / Yaw：拖动滑块控制姿态角度。
例如：拖动 Roll 到 10 度，无人机应向右倾斜并保持。
2.3 紧急停止 (Emergency Stop)
方法：直接取消勾选 [2] 允许解锁电机。
效果：软件发送强制上锁指令，飞控切回 MANUAL 模式，电机立即停转。

 第三章：AI 全自动 PID 整定 (实战)

3.1 准备工作
点击顶部导航栏切换到 ⚙ PID 智能整定 页面。
填写物理参数（顶部）：
惯量 (J)：台架转动惯量。小机架填 0.05，F450/F330 填 0.15，大机架填 0.25。此参数决定了 AI 调节的“力度”。
让飞机“飞”起来：
回到控制台，解锁电机。
拉高油门 (THR) 至悬停点（约 1200~1600），确保绳索拉紧或台架受力，无人机处于“有劲儿”的状态。（非常重要！怠速状态下整定是无效的）
3.2 开始整定
选择目标轴：推荐先整定 Roll 或 Pitch（通常这两个轴参数接近）。
点击 [▷ 开始自动整定]。
观察现象：
无人机将自动执行一连串的阶跃信号（左右晃动）。
右侧曲线图会实时画出：
白虚线：目标角度。
红线：整定前的抽搐波形。
绿线：优化后的丝滑波形。
查看诊断报告：
下方文本框会实时输出 AI 诊断结果，例如：
[🔴 内环高频发散] ➡️ AI 自动削减了 D 项。
[🔸 外环逼迫过猛] ➡️ AI 降低了外环 P。
[🟡 稳态静差阻滞] ➡️ AI 增加了积分 I。
结束：当满意时，点击 [□ 停止整定]。此时最新参数已自动写入飞控。
📊 第四章：阶跃性能测试 (科研评估)
整定完 PID 后，如何科学地证明“效果好”？请使用阶跃测试功能。

4.1 测试流程
保持悬停油门：确保电机依然处于解锁且有推力的状态。
设置幅度：在“阶跃测试幅度”框中输入角度，推荐 15 度。
点击 [⚡ 运行阶跃性能测试]。
动作过程：
无人机归零 ➡️ 突变至 15 度 ➡️ 保持 2 秒 ➡️ 归零。
获取成绩单：
测试结束后，左下角 “真实性能指标” 面板会自动刷新数据：
超调量 (Overshoot)：理想应 < 10%。
上升时间 (Rise Time)：反映响应速度，越快越好。
稳态误差 (Steady Error)：理想应 < 1 度。
💾 第五章：数据闭环与工程管理
5.1 数据导出
科研绘图：点击右上角 [↓ 导出数据]。
生成 CSV 文件，包含：时间戳、三轴角度、三轴目标值、4路电机PWM。
可直接导入 MATLAB / Python / Origin 绘制高水平论文图表。
5.2 参数管理
导出参数 (JSON)：保存当前调好的 PID 参数，作为“黄金存档”。
后悔药 (Undo)：整定歪了？点击左上角 [↩ 撤销回滚]，一键退回上一次的状态。
恢复出厂：点击右下角 [撤销修改(恢复初值)]，强行恢复到软件刚连接时的状态。
❓ 常见问题 (Q&A)
Q1: 点击“启动连接”没反应？

A: 检查 USB 线是否插好，或者 RflySim 是否已开启 CopterSim。
Q2: 解锁后电机不转？

A: 检查电池电压；检查是否设置了“机头方向”错误导致飞控保护；检查 PWM 输出值（右下角柱状图）是否有变化。
Q3: 自动整定时飞机晃动太剧烈！

A: 立即取消勾选 [1] 允许发送指令 切断信号！然后减小“整定激进程度”到“保守”，或手动降低一点 P 值再试。
Q4: 阶跃测试的数据全是 0？

A: 你可能忘了拉高油门。如果无人机在地上趴着（油门=0），它是无法响应姿态指令的。请推 THR 滑块到 1200+。
